# 클래스101 프론트앤드 개발자 과제

안녕하세요. 클래스101 프론트앤드 직군에 지원한 **이윤혜** 입니다.    
해당 과제는 `React`로 개발하였습니다.

## 사용 라이브러리

- 상태관리 : `mobx`
- UI : `material-ui`, `react-toastify`
- `typescript`

## 실행방법
```bash
npm run start
```
혹은
```bash
yarn start
```
커맨드를 입력하시면 [localhost:3001](http://localhost:3001/) 에서 확인하실 수 있습니다.
